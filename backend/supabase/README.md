# Supabase Backend

โฟลเดอร์นี้เก็บ backend logic ของ Supabase:

- schema changes
- RLS policies
- indexes
- RPC functions
- realtime publication setup

ให้รันไฟล์ SQL ใน `migrations/` ผ่าน Supabase SQL Editor หรือเปลี่ยนไปใช้ Supabase CLI migration ในอนาคต

```txt
backend/supabase/migrations/
└─ 001_win_queue_logic.sql
```
